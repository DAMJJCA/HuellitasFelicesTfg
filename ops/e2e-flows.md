# Flujos E2E recomendados

## Agenda clinica

1. Login como recepcion.
2. Crear una cita futura para una mascota.
3. Confirmar la cita.
4. Login como veterinario.
5. Iniciar consulta desde la cita.
6. Completar diagnostico y finalizar.
7. Abrir informe imprimible.

## Documentacion clinica

1. Login como auxiliar.
2. Abrir documentos medicos.
3. Subir un PDF asociado a una mascota.
4. Verificar que aparece en historial.
5. Login como cliente.
6. Verificar que solo ve sus documentos.

## Facturacion

1. Login como recepcion.
2. Crear factura para un cliente.
3. Emitir factura.
4. Marcar como pagada.
5. Verificar importes y estado en la tabla.

## Permisos

1. Recepcion puede gestionar agenda, clientes, disponibilidad y facturacion.
2. Auxiliar puede ver agenda y gestionar documentos/preventivos, pero no finalizar consultas.
3. Cliente solo ve su informacion.
4. Veterinario no crea clientes ni mascotas, pero trabaja consultas.
